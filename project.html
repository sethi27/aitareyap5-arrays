Interactive Particle Animation

An interactive particle system demonstrating array operations, loops, and particle physics. Create and animate particles in various patterns and colors.

How to Use

  • Click and drag to create random particles
  • Use shape buttons to add specific particles
  • Press 'C' to cycle through color schemes
  • Select a pattern and click "Start Loop" to animate
  • Use "Clear All" to reset

Technical Features

  • Array operations (push, splice)
  • For loops for particle updates
  • Object-oriented particle system
  • Interactive DOM elements
  • P5.js animation framework
